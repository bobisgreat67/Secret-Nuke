## 🛠️ Troubleshooting

### Problem: App won't open
- Run as administrator (right-click → Run as administrator)
- Make sure Windows Defender isn't blocking it

### Problem: "Failed to connect" error
- Check your internet connection
- The bot might be offline - wait a few minutes
- Make sure you're not rate limited

### Problem: "Missing permissions" error
- The bot needs ADMIN permission in the server
- Make sure the bot is in the server
- Bot must have "Administrator" permission

### Problem: Features aren't working
- You may be rate limited
- Wait 5-10 minutes and try again
- Some features may be disabled by Discord

### Problem: "Server not found" error
- Make sure you selected a server from the list
- The bot must be in that server
- Reload the server list

## 🔧 Features Explained

### Nuke Server
- Deletes all channels
- Deletes all roles
- Deletes all emojis
- Kicks all members
- Renames the server
- Completely destroys the server

### Reverse Nuke
- Restores server from backup
- Brings back channels
- Restores roles and permissions
- Creates everything back

### Kick All
- Kicks every member from the server
- Works even on members with higher roles
- Bot must have "Kick Members" permission

### Backup
- Saves server structure
- Saves all channels and roles
- Can be used to restore later
